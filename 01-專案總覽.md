# 專案總覽

## 定位

NVIDIA NemoClaw 是一套開源參考堆疊，用於在 OpenShell 沙箱中安全地運行 OpenClaw 常駐助理，並整合 NVIDIA 推論後端（Nemotron 模型，經由 NVIDIA Endpoint API 或本地 NIM/vLLM/Ollama）。

核心價值：

- 以宣告式策略控制網路出口、檔案系統存取與權限提升
- 將 AI 代理運行在受隔離的容器沙箱中，降低安全風險
- 支援多種推論後端，從雲端 API 到本地 GPU 推論無縫切換

## 版本資訊

- 目前版本：`0.1.0`（Alpha，2026 年 3 月 16 日發佈）
- 授權條款：Apache-2.0
- 最低需求：Node.js 22+、Python 3.11+、Docker

## 技術棧

| 層級 | 語言 | 主要框架/工具 |
|------|------|---------------|
| CLI 入口點 | Node.js (ES Modules) | 無外部相依 |
| TypeScript 插件 | TypeScript (ES2022) | Commander、tar、json5、yaml |
| Python Blueprint | Python 3.11+ | PyYAML、uv（套件管理） |
| 容器沙箱 | Docker | OpenShell 隔離運行環境 |
| 測試 | JavaScript/TypeScript | Vitest 4.1 |
| 程式碼品質 | — | ESLint、Prettier、Ruff、Pyright、prek |

## 關鍵檔案路徑

| 用途 | 檔案路徑 |
|------|----------|
| 插件狀態 | `~/.nemoclaw/state/nemoclaw.json` |
| Onboard 設定 | `~/.nemoclaw/config.json` |
| 憑證儲存 | `~/.nemoclaw/credentials.json`（權限 0o600） |
| 沙箱註冊表 | `~/.nemoclaw/sandboxes.json`（權限 0o600） |
| 遷移快照 | `~/.nemoclaw/snapshots/{timestamp}/` |
| 執行狀態 | `~/.nemoclaw/state/runs/{run_id}/plan.json` |
| OpenClaw 設定 | `~/.openclaw/openclaw.json` |
| OpenClaw 工作區 | `~/.openclaw/workspace/` |
